중국어 간체 → 원어민 발음 (브라우저 내장 TTS) 문장을 입력하면 브라우저의 Speech Synthesis를 사용해 중국어(표준어)로 읽어줍니다. GitHub Pages에서 그대로 동작하며, 서버가 필요 없습니다. 중국어 간체 문장들 음성/목소리 속도 1.0x 피치 1.0 원어민처럼 읽기 정지 녹음하여 파일로 저장 (실험적) ※ 쉼표(，)는 짧은 쉬는 타이밍으로, 마침표(。)、물음표（？）、느낌표（！） 기준으로 문장을 분리합니다. 대기열(문장별) 문장을 입력하면 여기에 분리되어 표시됩니다. • 중국어 음성이 보이지 않으면: OS/브라우저에 중국어 음성이 설치되어 있어야 합니다. • 파일 저장: 브라우저가 탭 오디오 공유를 지원할 때만 동작합니다(크롬/엣지 권장).
